# 股票量化系统 MVP 方案

## MVP 核心功能

### 初始股票池
先不要覆盖全市场，建议从：

沪深 A 股主板 + 创业板 + 科创板
剔除 ST、退市风险、长期停牌
剔除上市不足 120 个交易日的新股
剔除日均成交额过低的股票，例如近 20 日平均成交额 < 5000 万

### 股票池管理
全市场股票池
├── 可交易股票池
│   ├── 主线题材池
│   ├── 业绩增长池
│   ├── 趋势强势池
│   ├── 低位反转池
│   └── 风险观察池


## 选股策略设计

### 选股总逻辑
基础过滤
→ 行情因子
→ 资金因子
→ 财务因子
→ 题材资讯因子
→ 风险因子
→ 综合评分
→ 股票评级
→ 虚拟买入/观察

## 因子体系

### 技术趋势因子，权重 30%
| 因子                | 说明     | 加分逻辑      |
| ----------------- | ------ | --------- |
| MA5 > MA10 > MA20 | 短期均线多头 | 加分        |
| 股价站上 MA20         | 趋势转强   | 加分        |
| 20 日涨幅            | 判断强度   | 适度加分，过高扣分 |
| 60 日新高            | 趋势突破   | 加分        |
| 放量上涨              | 量价配合   | 加分        |
| 缩量回踩              | 强势整理   | 加分        |

#### 建议规则：
##### 趋势分 = 0 ~ 30

+5  股价 > MA20
+5  MA5 > MA10 > MA20
+5  最近 20 日涨幅在 5% ~ 35%
+5  最近 10 日有放量阳线
+5  距离 60 日新高小于 5%
+5  回撤小于 15%
-5  跌破 MA20
-5  最近 20 日涨幅 > 60%，短线过热
-5  最近 5 日放量下跌
-10 股价跌破 MA60

##### 扣分
-5  跌破 MA20
-5  最近 20 日涨幅 > 60%，短线过热
-5  最近 5 日放量下跌
-10 股价跌破 MA60


### 资金强度因子，权重 25%
| 因子    | 说明          |
| ----- | ----------- |
| 成交额排名 | 判断资金关注度     |
| 换手率   | 判断活跃度       |
| 主力净流入 | 可参考，但不要过度依赖 |
| 北向资金  | 适合大票        |
| 龙虎榜   | 适合短线题材股     |

#### 建议规则：

##### 资金分 = 0 ~ 25

+5  近 5 日成交额持续放大
+5  今日成交额大于近 20 日均值 1.5 倍
+5  换手率在 3% ~ 15%
+5  近 3 日资金净流入为正
+5  所属板块成交额排名靠前

##### 扣分
-5  高位爆量长上影
-5  连续 3 日资金流出
-5  换手率过高，比如 > 30%

### 基本面因子，权重 20%
| 因子      | 说明     |
| ------- | ------ |
| 营收同比增长  | 判断成长性  |
| 净利润同比增长 | 判断盈利能力 |
| 毛利率变化   | 判断竞争力  |
| 资产负债率   | 风险控制   |
| ROE     | 盈利质量   |

#### 建议规则：
##### 基本面分 = 0 ~ 20

+5  营收同比增长 > 10%
+5  净利润同比增长 > 10%
+5  ROE > 8%
+3  毛利率稳定或提升
+2  资产负债率不过高

##### 扣分

-5  净利润连续下滑
-5  扣非净利润为负
-5  商誉占比过高
-5  资产负债率明显高于行业平均

### 题材资讯因子，权重 15%

#### 资讯分不建议完全交给大模型主观判断，而是：
关键词匹配 + 事件分类 + 情绪判断 + 可信源权重

#### 资讯类型
政策利好
产业趋势
订单合同
业绩预增
涨价逻辑
技术突破
机构调研
高管增持
减持风险
监管风险
澄清公告
业绩暴雷

#### 资讯评分
##### 资讯分 = -15 ~ +15

+5  公司公告出现订单、业绩预增、回购、增持
+5  所属行业出现政策或产业利好
+3  多家媒体报道同一正面事件
+2  机构调研热度上升

-5  减持、监管问询、诉讼
-5  业绩预亏、商誉减值
-3  高位澄清风险
-3  负面舆情增多

### 风险因子，权重 -20 到 0
风险因子不是加分，而是扣分。

#### 风险分 = 0 ~ -20

-5   ST / 退市风险 / 重大诉讼
-5   近 20 日涨幅过高且无业绩支撑
-5   大股东减持
-5   连续跌破关键均线
-5   业绩亏损扩大
-5   换手率异常过高
-10  监管处罚 / 财务造假风险

## 综合评分与评级
### 总分公式
总分 = 趋势分 * 30%
     + 资金分 * 25%
     + 基本面分 * 20%
     + 资讯分 * 15%
     + 风险调整分

为了工程实现简单，可以统一换算成 100 分制。
Score = TrendScore + MoneyScore + FundamentalScore + NewsScore + RiskScore

### 评级体系
|       分数 | 评级 | 含义              | 操作建议        |
| -------: | -- | --------------- | ----------- |
| 85 - 100 | S  | 强趋势 + 强资金 + 强题材 | 重点观察 / 虚拟买入 |
|  75 - 84 | A  | 有较强机会           | 加入核心观察池     |
|  65 - 74 | B  | 有潜力但确认不足        | 观察          |
|  50 - 64 | C  | 普通              | 不操作         |
|     < 50 | D  | 弱势或风险较大         | 剔除          |
|     风险触发 | R  | 风险票             | 禁止买入        |
